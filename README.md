Ця програма моделює роботу домашніх електроприладів. У проєкті реалізовано ієрархію приладів (кухонні, кліматичні, розважальні, прибиральні). Користувач через мінімальне консольне меню може переглядати список приладів, вмикати їх, підраховувати загальну споживану потужність, сортувати за потужністю та знаходити прилади за вказаним діапазоном.
